# Privacy Policy

Smart Field Log Generator is a Google Workspace add-on that helps field teams create structured, standardized log entries in Google Docs.

- **Data Access**: The add-on reads form data entered by the user and creates Google Docs documents. No external data is accessed.
- **Data Storage**: Log documents are stored in Google Drive. No data is stored in external systems.
- **Data Sharing**: No data is sent to or shared with third parties. All processing happens within Google's infrastructure.
- **File Operations**: The add-on creates new Google Docs documents based on form input.
- **Logging**: Operational logs are written to Apps Script Logger for troubleshooting.
- **Removal**: Remove stored templates and settings by uninstalling the add-on or clearing Script Properties.
- **Support**: For privacy questions, open an issue in the repository or contact support@tangentforge.com.
